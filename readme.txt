== Description ==

**Stylish Twitter Profile Box** is WordPress widget plugin, adds a stylish and responsive profile box on sidebars, footer's widget area, displaying the twitter profile picture, twitter name, username, follow button and twitter meta counts.
Bloggers, content marketers, brands can get more exposure, identity and engagement for their readers, users and customers.


= Main Features =

* Shows twitter profile picture, name, username, follow button and profile meta counts
* Eye Catching Styles (Never slip your users)
* Latest Twitter API
* Responsive and Cross-broswers support
* Custom timing to save data in Cache.
For more features donwload [pro version](http://stylishtwitterprofilebox.com).
 
== Installation ==

1. Download the plugin (.zip file) on your hard drive.
2. Unzip the zip file contents.
3. Upload the `stylish-twitter-profile-box` folder to the `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. A new widget `Stylish Twitter Profile Box` will appear in Appearance > Widgets.
6. Drag and drop `Stylish Twitter Profile Box` widget to sidebar or footer widget.

If you need more info, please read the plugin [documentation](http://stylishtwitterprofilebox.com/docs/).
